About the Blaeu Dashboard

This dashboard lets users browse, search and filter Bob Henery's transcription of the Blaeu Atlas of Scotland (1654), which is hosted by the NLS as five separate lists:

A-D |   E-H   |   I-M   |   N-Q   |   R-Z

Features:

  • Browse 28,000+ transcribed place names
  • Search by historical or modern name
  • Filter by map or feature type
  • View locations on original atlas maps
  • Explore statistics
